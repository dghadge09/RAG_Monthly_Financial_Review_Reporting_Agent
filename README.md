# RAG_Monthly_Financial_Review_Reporting_Agent
# Monthly Financial Report Generation Workflow

## Overview
This Zapier workflow automates the generation and delivery of monthly financial reports. It triggers on the first day of each month, analyzes financial metrics using a vector database, and emails a professionally formatted report to stakeholders.

## What It Does
1. **Triggers automatically** on the 1st of every month at 8:00 AM
2. **Queries financial data** from the NovaCart Vector DB Agent
3. **Enriches analysis** using knowledge sources (semiconductor, tech trends, and DI tech reports)
4. **Generates a comprehensive report** with key metrics:
   - Net Profit Margin
   - Revenue Growth Rate
   - Return on Investment (ROI)
   - Cash Flow
   - Trends and strategic recommendations
5. **Delivers via email** as a professionally formatted HTML report

## Key Features
- ✅ Scheduled monthly automation (no manual intervention)
- ✅ Vector database integration for intelligent data retrieval
- ✅ Knowledge-enhanced financial analysis with external context
- ✅ Professional HTML email formatting
- ✅ Dynamic report dates using template variables
- ✅ Comprehensive financial metrics dashboard

## Workflow Role
**Senior Financial Analyst** – Generates insightful business performance reports backed by data analysis and industry insights.
